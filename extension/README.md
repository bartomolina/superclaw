# SuperClaw Extension

Browser extension for sending UI feedback into SuperClaw Kanban.

## Local build

```bash
pnpm install
pnpm build
```

Build output:
- `.output/chrome-mv3/`

Optional zip:

```bash
pnpm zip
```

## Notes

- It connects to the local Kanban app.
- Install it as an unpacked extension or from a built artifact.
